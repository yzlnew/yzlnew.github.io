在 VPS 部署 TeamSpeak 语音服务器一键脚本

运行要求

  • Ubuntu14+ 64bit
  • sudo

使用方法

wget https://raw.githubusercontent.com/yzlnew/TeamSpeak3Setup/master/ts3setup.sh
chmod +x ./ts3setup.sh
